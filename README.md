## WinfxkLib 是什么?

一个[PowerNukkit](https://github.com/PowerNukkit/PowerNukkit)的插件开发运行库
#### 您是否需要[立即下载WinfxkLib？](https://github.com/WinfXK/WinfxkLib/releases)


## WinfxkLib 有哪些功能？
> - 全自动更新(为所有基于WinfxkLib的插件检查更新)
> - 语言文件解析，完全的可自定义插件文本
> - 物品的存储及其解析
> - 全套的物品-ID-图标对应信息
> - 全套的GUI管理及API
> - 经济支持管理，方便兼容各个经济插件
> - 插件配置文件管理，自动生成系统默认配置或修复已经损坏的配置
